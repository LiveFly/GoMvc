程序的目录结构十分简单，如下所示： 
└── src                                          
    ├──System                                 // GoMvc系统文件所在的目录
    ├── Controllers                          // 所有Controller都在这个目录下
    ├── Model                                //所有模型层的代码放在此处
    ├── Static                               //所有的静态文件放在此处，如JS，CSS，图片等静态资源，静态目录可以在配置文件中配置
    └── Views                                //所有的视图页面存放在这里，文件的扩展名为.ghtm
