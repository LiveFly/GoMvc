GoMvc提供了一个日志类，其代码位于"System/Log/Logger.go"文件中。提供了Add(string),AddError(error)两个接口。 日志的保存格式为"年\月\年-月-日.log"。其存放位置、大小可在配置文件中配置。配置文件修改后可立即生效，在"System/Web/WebApplication.go"有对配置文件进行监控，当配置文件改变后，立即重新加载。

     在程序中要想使用使用日志功能有两种方法。

  1，引入System/Web 包，使用Web.App.Log来记录日志，示例如下：

import (
	"System/Web"
)
func  TestLog() {
	Web.App.Log.Add("日志")
}

2，引入System/Log，使用AppLog来记录日志，示例如下：

import (
	"System/Log"
)
func  TestLog() {
	Log.AppLog.Add("日志")
}
